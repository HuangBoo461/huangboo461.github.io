记录github博客开启gitalk

gitalk： 一个基于 Github Issue 和 Preact 开发的评论插件,方便用户交流

准备工作

注册github OAuth Apps 链接

点击注册后跳转到成功页面

修改主题下配置文件(netx/_config.yml)

至此gitalk开启成功。

此时博客页面需要登录github初始化才行，并且每篇博客都要重新初始化，下篇介绍如何自动初始化。

参考地址 https://asdfv1929.github.io/2018/01/20/gitalk/

注:参考以上地址开启gitalk时，发现使用最新next主题只需要配置即可。
